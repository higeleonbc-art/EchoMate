EchoMate アバター立ち絵 素材フォルダ
======================================

このフォルダに以下のファイル名で PNG 画像（RGBA 推奨）を配置すると
アバターウィンドウに立ち絵が表示されます。

【ベース画像】 ─ 目は描かずに口の開閉差分を用意してください
  base_idle_close.png  : 通常表情・口閉じ
  base_idle_open.png   : 通常表情・口開き（喋り中）
  base_smile_close.png : 笑顔・口閉じ（高テンション時）
  base_smile_open.png  : 笑顔・口開き（高テンション時・喋り中）

【目画像】 ─ ベースに重ねてアルファ合成されます
  eyes_open.png   : 目開き（通常）
  eyes_half.png   : 半目（まばたきの途中）
  eyes_close.png  : 目閉じ（まばたき）
  eyes_smile.png  : 笑い目（高テンション時）

【仕様】
  - 推奨サイズ: 300x400px 以上（ベースと目は同じサイズ推奨）
  - 形式: PNG (RGBA / 透過あり)
  - ベース画像と目画像は同じキャンバスサイズにしてください
  - サイズが異なる場合は自動リサイズされます

【アニメーション動作】
  - まばたき  : 3〜6秒おきにランダムで自動実行
  - 口パク    : VOICEVOX 音声再生中に口が開きます
  - 跳ねる    : テンション 0.6 以上（興奮・喜び）
  - 震える    : テンション 0.3 未満（緊張・落ち込み）
