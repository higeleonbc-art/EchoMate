# EchoMate 開発ロードマップ（Claude Code 連携用）

## 【1. 実装完了】完了済みタスク
以下の機能および修正はすべて実装・検証済みです。

### 基本機能・UX（v1〜v2）
1.  **会話の動的付与**: 質問・掘り下げによる自発的対話。
2.  **フィラー先行再生**: 生成待ち時間を「あー」「なるほど」等で埋める。
3.  **再生キャンセル (Barge-in)**: プレイヤー発言時の AI 音声停止。
4.  **記憶・成長システム**: 過去の話題再利用、親密度、プレイスタイル反映。
5.  **感情別ボイス**: テンションに連動した VOICEVOX スタイル切り替え。
6.  **Webソース学習**: URL からのゲーム知識抽出と RAG 統合。

### システム近代化・バグ修正（2026-04-17 実施）
7.  **通信ライブラリ刷新**: 全通信を `requests` から `httpx` (タイムアウト厳格化) へ移行。
8.  **環境管理**: `.env` によるモデル名（`qwen3:8b` 等）と API URL の一括管理。
9.  **音声エンジン安定化**: `pyaudiowpatch` 導入による Windows ループバック対応。
10. **ログ管理**: 10MB/5世代のローテーション実装（`echomate.log` の肥大化防止）。
11. **AI 応答の正常化**: 制約（開始語等）のプロンプト注入により、検証失敗による定型文落下問題を解消。
12. **イベント反応の AI 化**: キル・デス時の反応をテンプレから LLM 生成へアップグレード。
