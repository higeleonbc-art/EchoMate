# Phase 3: 人間らしい気配（操作量検知）と VLM の深化（実施中）

## 【1】プレイヤー操作の検知（集中度・パニック度の可視化）
- **目的**: AI に「プレイヤーがどれだけ激しく操作しているか」を教え、会話のきっかけにする。
- **実装手順**:
    1. `pynput` をインストール (`pip install pynput`)。
    2. `input_monitor.py` を新規作成し、マウス・キーボードのイベントをカウントするクラスを実装。
    3. `state_manager.py` の `PlayerState` に `input_intensity` (0.0~1.0) を追加。
    4. `StateManager.tick()` 内で、秒間操作数に基づく強度の算出と指数移動平均による平滑化を実装。
    5. `ai.py` のプロンプトに「操作の激しさ」を注入。

## 【2】VLM (Vision Language Model) の深化（自発的な画面観察）
- **目的**: 音トリガーだけでなく、自発的に画面を「観察」して話しかけるようにする。
- **実装手順**:
    1. `main.py` に `PROACTIVE_VISION_INTERVAL` (20秒程度) を導入。
    2. `_proactive_loop` 内で、一定時間ごとに VLM に「今の画面状況を簡潔に教えて」とリクエストする。
    3. 解析結果を `ai.py` のコンテキストに保持し、「さっきと比べてどう変わったか」を LLM が判断できるようにする。

---
## 修正が必要なコード詳細（メモ）

### `state_manager.py`
- `PlayerState` に `input_intensity` フィールドを追加。
- `input_label`（ほぼ静止 / 安定 / 激しい / 非常に激しい）の定義。
- `record_input_event()` メソッドの実装。

### `ai.py`
- `summary()` メソッドで操作強度を文字列化してプロンプトに含める。
- 操作強度が高い場合に「忙しそうだね」「連打えぐいｗ」などの応答を誘発する命令の追加。
- VLM の解析結果を履歴として保持し、推移に触れられるようにする。

### `main.py`
- `InputMonitor` の初期化と `state_manager` への接続。
- 自発的 VLM 解析タイマーの管理ロジック。

## 【追加修正】OpenCV 判定ロジックの拡張 (LoL 対応)
- **目的**: LoL のように「バーが短くなる（特定の色が減る）」現象を検知するため、比率の「以下」判定をサポートする。
- **実装手順**:
    1. `opencv_detector.py` の `_detect_by_color` を修正。
    2. `params` から `operator` ("ge" or "le") を取得し、判定に使用。
    3. `cv_config.json` を `green` かつ `le` (以下) に更新。

# League of Legends 用 `cv_config.json` 設定テンプレート (1600x900 / UI最小)

以下の内容を `cv_config.json` に手動で適用してください。

```json
{
  "_generated_for": "League of Legends (1600x900 Min-UI)",
  "zones": [
    {
      "name": "hp_bar_low",
      "region": {
        "left": 710,
        "top": 855,
        "width": 180,
        "height": 15
      },
      "event_type": "low_hp",
      "method": "color_threshold",
      "params": {
        "color": "green",
        "operator": "le",
        "threshold": 0.20
      },
      "cooldown": 5.0,
      "min_hits": 4,
      "enabled": true
    },
    {
      "name": "kill_feed",
      "region": {
        "left": 1200,
        "top": 40,
        "width": 380,
        "height": 250
      },
      "event_type": "kill",
      "method": "brightness",
      "params": {
        "mode": "spike",
        "threshold": 45
      },
      "cooldown": 3.0,
      "enabled": true
    }
  ],
  "audio_rules": [
    {

      "threshold": 0.4,
```

> 画面上の座標は環境（1080p, 4Kなど）や HUD サイズによって異なる場合があります。
